Android

Required Files:

Installation Guide:

  1. Download both the APK and OBB files by clicking the buttons.
  2. Install the KIPAS_PRIVATE_SERVER.apk file. Do not open it yet.
  3. Extract the com.dts.freefiremax.zip file using ZArchiver or a similar app.
  4. You will get a folder named com.dts.freefiremax.
  5. Move this folder to the following directory: Internal Storage/Android/obb/
  6. Done! You can now open and play the game.

iOS

Required Files:

Get Proxypin (AppStore)

Installation Guide:

  1. Install the Proxypin application from the AppStore.
  2. Download the Config file (ffkipasfree.config) from the link.
  3. Open the Proxypin app, then import the downloaded config file.
  4. Activate the configuration within Proxypin.
  5. Done! Open your game.

Emulator (ROOT Required)

Required Files:

Get Proxypin (PlayStore)

Installation Guide:

  1. Ensure your Emulator (e.g., BlueStacks, LDPlayer) has ROOT access enabled.
  2. Install the Proxypin application from the PlayStore inside the emulator.
  3. Download the Config file (ffkipasfreez.config) from the link.
  4. Open the Proxypin app, then import the downloaded config file.
  5. Activate the configuration within Proxypin.
  6. Done! Open your game.
